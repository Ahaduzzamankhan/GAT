# GAT - Game Activity Tracker

GAT (Game Activity Tracker) is a modern Windows desktop application designed to automatically detect, monitor, and record gaming activity in real time. Built with Electron, React, TypeScript, and a high-performance Go backend, GAT provides accurate game tracking while maintaining low system resource usage.

The application records play sessions, total playtime, activity history, and game statistics, allowing users to understand their gaming habits through a clean and responsive interface. GAT can automatically identify supported games, store session data in a local SQLite database, and retrieve metadata such as game titles and artwork for a richer experience.

Features include real-time game detection, detailed playtime statistics, favorites management, export options for JSON and CSV, optional desktop notifications, system tray integration, customizable settings, and automatic metadata caching. The architecture separates the user interface, backend services, and Electron processes, improving reliability, maintainability, and performance.

Whether you want to track your gaming sessions, analyze playtime trends, or build a personal gaming history library, GAT offers a lightweight and efficient solution for Windows users.
